## StudyTracker: Web App de Controle de Estudos

Aplicação web para registro e acompanhamento de tempo de estudo com login de usuário.

## Funcionalidades:

- Login de usuário
- Timer de estudo
- Registro de sessões (SQLite)
- Interface responsiva com CSS e Bootstrap

## Tecnologias: 

Python | Flask | SQLite | HTML | CSS | Bootstrap | Git | GitHub
